# Olist Logistics Audit


 ## Project Overview

This project analyzes delivery performance in the Olist Brazilian E-commerce dataset.

The goal is to identify:
1. delivery delay patterns
2. customer satisfaction impact
3. eographic logistics issues
4. business insights from shipping performance

The analysis was performed using Python, Pandas, Matplotlib, and Jupyter Notebook.

## Key Findings

- Approximately 90% of deliveries arrived on time.
- Around 9.5% of orders experienced delays.
- Super-late deliveries had the worst customer satisfaction scores.
- Delivery delays strongly reduced review ratings.
- States such as SP and RJ recorded the highest number of delayed deliveries.
